# Instructions for Coding Agents

## Working Documents

All working documents go in `.scratch/`. This directory is gitignored and never committed. 

## Committed Artifacts

In source code, README, CLAUDE.md, and commit messages:
- describe only what the code is and what it does.
- do not mention planning docs, reference docs, or tracking docs.

## Git Workflow

- Commit working code as you go
- `.scratch/` is gitignored
